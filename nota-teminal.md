## comandos usados
# 1- pwd
Su funcion es mostrar la carpeta donde uno se encuentra. Antes de crear los archivos use esta funcion para asegurarme de estar en la carpeta correcta.
# 2- ls
Este comando lista todos los archivos de la carpeta actual. Use este comando para ver si el archivo se creo correctamente en la carpeta.
# 3- cd
Cd permite navegar de carpeta en carpeta. Navegue entre los archivos creados usando este comando.
# 4- mkdir
Este comando crea una nueva carpeta. Con el cree la carpeta tp1-mi-sitio.
# 5- touch
Su funcion es crear un archivo que se encuentra vacio. Con el cree el archivo index.html .
# 6- cat
Cat muestra el contenido de un archivo. Lo use para ver el contenido de mis archivos.
# 7- which
Este comando muestra la ruta completa de un ejecutable o comando. Lo use para ver donde se instalo Git.
# 8- code
El comando code abre VS Code desde la terminal. Con el abri mi proyecto.
# 9- clear
Este comando limpia la terminal. Lo use para trabajar de manera mas organizada.
# 10- history
Muestra el historial de comandos. Lo utilice para ver los comando usados.
